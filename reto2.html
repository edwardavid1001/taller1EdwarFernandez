Evento Descripción Ejemplo
click Se dispara cuando se hace clic en un elemento
change Se dispara cuando el valor de un elemento cambia (por ejemplo, un campo de entrada)
mouseover Se dispara cuando el puntero del mouse se mueve sobre un elemento
keydown Se dispara cuando una tecla se presiona en el teclado
focus Se dispara cuando un elemento recibe el foco (seleccionado o enfocado)
blur Se dispara cuando un elemento pierde el foco
dblclick Se dispara cuando se hace doble clic en un elemento
dragstart Se dispara cuando se inicia el arrastre de un elemento
scroll Se dispara cuando se desplaza el contenido de un elemento
